# Output Format

Choose the deliverable by the requested mode.

## Summary

State:

- product, audience, geography, and goal;
- sources actually used;
- key assumptions and missing data;
- highest-priority clusters and main exclusions.

## Cluster Table

```markdown
| Cluster | Intent | Demand stage | Representative phrases | Landing page | Priority | Notes |
|---|---|---|---|---|---|---|
```

For advertising also include:

- recommended campaign and ad-group boundary;
- group/campaign negative keywords with reasons;
- brand/competitor separation;
- what to validate after launch.

For SEO also include:

- recommended page or section;
- page type: category, service, product, article, FAQ;
- cannibalization risks;
- internal-linking relationship.

## Evidence Rules

- Attach frequency only to the exact query/region/period returned by Wordstat.
- Mark generated expansions and estimates explicitly.
- Do not present rearranged phrases as measured demand.
- If the full core is large, save it to a file or connected sheet and show a concise summary in chat.
