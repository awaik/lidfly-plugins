# Human Editorial Polish

Use when text sounds generic, over-structured, or AI-like and needs a natural editorial pass.

## Goal

Make the text clearer, more specific, and more human without lowering quality or changing facts.

## Rules

- Preserve meaning, facts, links, headings, tables, and formatting.
- Remove cliches such as "в современном мире", "важно отметить", "следует подчеркнуть".
- Replace vague claims with concrete wording already supported by the text.
- Vary sentence rhythm naturally.
- Keep grammar and spelling clean.
- Do not add intentional mistakes, false uncertainty, slang, or detector-bypass artifacts.

## Output

Return edited text or write a new final file if a path was provided. Keep the original unless the user explicitly asks to overwrite.
